数据库

知识点

  • MySQL:
    • 表设计：tb_question（字段：question_id, title, content, space_limit, time_limit）。
    • 索引优化：为 question_id 创建主键索引，提升查询效率。
    • 事务管理：确保提交记录和判题结果的一致性。
  • Redis:
    • 缓存热点题目：使用 String 类型存储题目详情（question:{id}）。
    • 会话管理：使用 Hash 类型存储用户会话（session:{userId}）。
    • TTL 机制：设置缓存过期时间（例如 1 小时）。

项目应用

  • 题目服务：从 MySQL 查询题目，缓存到 Redis。
  • 用户服务：使用 Redis 存储用户登录会话。
