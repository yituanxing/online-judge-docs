组件

知识点

  • Nacos:
    • 服务注册：各微服务注册到 Nacos。
    • 配置中心：管理微服务配置（例如数据库连接）。
  • RabbitMQ:
    • 异步处理：判题任务通过消息队列分发。
  • Elasticsearch:
    • 题目搜索：支持分词（例如“正整数的和”）。
  • XXL-JOB:
    • 定时任务：清理过期数据（结合 Redis TTL）。
  • 阿里云短信服务:
    • 用户注册：发送验证码。
  • ELK 栈:
    • 日志收集：Elasticsearch 存储，Kibana 展示。
  • Monaco Editor:
    • 代码编辑器：前端集成，支持 Java 语法高亮。

项目应用

  • 判题服务：使用 RabbitMQ 和 XXL-JOB 实现异步和定时任务。
  • 用户服务：集成阿里云短信服务。
  • 前端：使用 Monaco Editor 提供代码编辑功能。
