用户指南

欢迎使用 Online Judge 系统！本指南将帮助你快速上手系统的 C 端（用户端） 和 B 端（管理端），分别面向普通用户和管理员。

C 端（用户端）使用指南

C 端是为普通用户设计的界面，主要用于刷题、查看题目列表、提交代码以及查看个人排名等功能。

界面概览

C 端首页以直观的布局展示主要功能，包含题目列表、个人排名、公告和日历等模块。

  • 顶部导航栏：包括“首页”、“题库”、“竞赛”等入口，右侧显示用户登录状态（例如“登录/注册”）。
  • 公告模块：显示系统公告，如“初试部署”和“竞赛部署成功”。
  • 题目列表：展示题目编号、标题、分类和难度，点击题目可进入答题页面。
  • 日历和排名：右侧显示日历和个人排名信息，方便用户查看活动和自己的排名。

使用步骤

  1. 登录/注册：

    • 点击右上角“登录/注册”，输入用户名和密码登录，或注册新账号。
    • 登录后，右上角会显示你的用户名（例如“管理员”）。

  2. 浏览题目：

    • 在“题目列表”中，选择感兴趣的题目（如“初试部署成功”）。
    • 点击“已解答”查看已完成题目，点击“未解答”筛选待解决题目。

  3. 提交代码：

    • 进入题目详情页后，编写代码并提交。
    • 系统会自动评测代码，并显示评测结果（例如“已通过”或“未通过”）。

  4. 查看排名：

    • 右侧“排行榜”模块显示你的排名和积分（例如“第 1 名，积分 50”）。

B 端（管理端）使用指南

B 端是为管理员设计的界面，用于管理用户、题目、竞赛和系统设置等功能。

界面概览

B 端界面以列表形式展示管理内容，方便管理员快速查看和操作。

用户管理

  • 用户列表：展示用户 ID、用户名、邮箱、积分等信息。
  • 操作选项：支持筛选用户（例如“创建日期”）、添加用户、编辑或删除用户。

题目管理

  • 题目列表：展示题目 ID、标题、分类、创建时间等信息。
  • 操作选项：支持筛选题目（例如“创建日期”）、添加题目、编辑或删除题目。

竞赛管理

  • 竞赛列表：展示竞赛 ID、标题、分类、开始时间等信息。
  • 操作选项：支持筛选竞赛、添加竞赛、编辑或删除竞赛。

使用步骤

  1. 登录管理端：

    • 使用管理员账号登录 B 端，登录后进入管理仪表盘。

  2. 用户管理：

    • 在左侧导航栏选择“用户管理”。
    • 查看用户列表，点击“添加用户”创建新用户，或点击“编辑/删除”修改用户信息。

  3. 题目管理：

    • 在左侧导航栏选择“题目管理”。
    • 查看题目列表，点击“添加题目”创建新题目，或点击“编辑/删除”修改题目内容。

  4. 竞赛管理：

    • 在左侧导航栏选择“竞赛管理”。
    • 查看竞赛列表，点击“添加竞赛”创建新竞赛，或点击“编辑/删除”修改竞赛信息。

  5. 查看日志：

    • 在左侧导航栏选择“系统管理” -> “日志”，查看系统操作记录。

注意事项

  • C 端用户：

    • 提交代码前，请确保代码符合题目要求，避免频繁提交导致系统压力。
    • 定期查看公告，了解系统更新或竞赛信息。

  • B 端管理员：

    • 添加题目或竞赛时，确保信息准确，避免影响用户体验。
    • 定期清理无效用户或题目，保持系统高效运行。
