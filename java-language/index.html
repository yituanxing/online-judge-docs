Java 语言

知识点

  • Java 17 特性:
    • 记录类（Records）：简化实体类定义，例如 Question 实体。
    • 密封类（Sealed Classes）：限制继承，用于判题服务的状态管理。
  • 多线程:
    • 使用 ExecutorService 管理判题任务线程池。
    • 结合 RabbitMQ 实现异步处理。
  • 异常处理:
    • 自定义异常类（如 JudgeException），处理判题错误。
  • 集合框架:
    • 使用 List 和 Map 存储题目数据和判题结果。

项目应用

  • 判题服务：使用 Java 17 的记录类定义判题结果（record JudgeResult(String status, long time)）。
  • 题目服务：使用 HashMap 缓存题目元数据。
