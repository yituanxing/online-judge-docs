操作系统

知识点

  • 进程管理:
    • 使用 Docker 沙箱隔离用户代码，限制 CPU 和内存。
    • 通过 pid 跟踪判题进程。
  • 文件系统:
    • 临时文件存储用户代码（/tmp/judge/）。
  • Linux 部署:
    • 在阿里云 ECS（CentOS）上部署微服务。
    • 使用 systemd 管理服务进程。

项目应用

  • 判题服务：在 Docker 容器中运行用户代码，限制资源。
  • 部署：使用 Docker Compose 管理微服务容器。
